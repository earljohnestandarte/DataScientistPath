Python Basics Lessons

This repository contains my personal learning materials as I study Python fundamentals. Each lesson includes explanations and code examples to build a strong foundation in Python.

Lessons Covered

Variables and Dynamic Typing
Understanding how Python handles variable assignment and dynamic data types.

Core Data Types
Exploring integers, floats, booleans, strings, lists, tuples, sets, and dictionaries.

"Pythonic" Strings (f-strings)
Learning modern string formatting using f-strings for cleaner and more readable code.

Basic Input/Output
Using print() for display and input() for user interaction.
